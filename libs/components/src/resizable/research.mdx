# Resizable Research

## Overview
Resizable component allows users to adjust the size of panels by dragging handles between them.
The component is fully accessible, supports keyboard navigation, and complies with the WAI-ARIA specification for window splitters.

## Research Links:
- [W3C WAI-ARIA Window Splitter Pattern](https://www.w3.org/WAI/ARIA/apg/patterns/windowsplitter/)
- [Corvu Resizable](https://corvu.dev/docs/primitives/resizable/)

## Behavior

The component consists of three main pieces:
- Root: Container that manages the resizable panels
- Panel: Individual resizable sections
- Handle: Draggable divider between panels

---

### States

- Default: Panels at their initial sizes
- Resizing: User is dragging a handle
- Collapsed: Panel is in collapsed state (if collapsible)
- Disabled: Resizing is prevented

---

## Keyboard Interactions:
- Left/Right: Resize horizontally
- Up/Down: Resize vertically
- Shift + Arrow: Move in larger increments
- Alt + Arrow: More precise movement
- Home: Collapse preceding panels
- End: Expand preceding panels
- Enter: Toggle collapse/expand (for collapsible panels)

## API:

### Root:
```typescript
interface ResizableRootProps {
  orientation?: 'horizontal' | 'vertical';
  disabled?: boolean;
  onChange$?: (sizes: { [key: string]: number }) => void;
}
```

### Panel:
```typescript
interface ResizablePanelProps {
  width?: number;
  height?: number;
  minWidth?: number;
  maxWidth?: number;
  minHeight?: number;
  maxHeight?: number;
  collapsible?: boolean;
  collapsed?: boolean;
  collapsedSize?: number;
  collapseThreshold?: number;
}
```

### Handle:
```typescript
interface ResizableHandleProps {
  onResizeStart$?: () => void;
  onResize$?: (delta: number) => void;
  onResizeEnd$?: () => void;
  step?: number;
}
```

## Important CSS:

```css
/* Required CSS */
[data-qds-resizable-root] {
  display: flex;
  width: 100%;
  height: 100%;
  position: relative;
}

[data-qds-resizable-root][data-orientation="horizontal"] {
  flex-direction: row;
}

[data-qds-resizable-root][data-orientation="vertical"] {
  flex-direction: column;
}

[data-qds-resizable-panel] {
  position: relative;
  display: flex;
  min-width: 0;
  min-height: 0;
}

[data-qds-resizable-handle] {
  position: relative;
  touch-action: none;
  user-select: none;
}

[data-qds-resizable-root][data-dragging="true"] {
  user-select: none;
}
```
