# Radio Group Research

## Overview
Radio Group allows users to select a single option from a given list. The component is fully accessible,
supports keyboard navigation, and complies with the WAI-ARIA specification for radio groups.

## Research Links:
- WAI-ARIA Radio Group Pattern (https://www.w3.org/WAI/ARIA/apg/patterns/radio/)
- Radix UI Radio Group (https://www.radix-ui.com/primitives/docs/components/radio-group)
- MUI Radio Group (https://mui.com/material-ui/react-radio-button/)
- Chakra UI Radio Group (https://chakra-ui.com/docs/components/radio)
- Mantine Radio Group (https://mantine.dev/core/radio-group/)

## Keyboard Interactions:
- Tab: Move focus to radio group
- Space/Enter: Select focused option
- ArrowDown/ArrowRight: Move focus to next option
- ArrowUp/ArrowLeft: Move focus to previous option
- Home: Move focus to first option
- End: Move focus to last option
